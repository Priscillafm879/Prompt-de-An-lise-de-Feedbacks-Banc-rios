# 📊 Análise de Feedbacks Bancários com IA

Este projeto foi desenvolvido como parte do **Desafio Criativo da DIO**, com o objetivo de criar um prompt estruturado para análise de feedbacks de clientes bancários utilizando Inteligência Artificial.

---

# 🎯 Objetivo do Projeto

O objetivo deste projeto é orientar uma IA a identificar:

- Temas recorrentes
- Reclamações frequentes
- Elogios dos clientes
- Sentimentos presentes nos feedbacks
- Oportunidades de melhoria
- Prioridades para ações corretivas

A proposta busca transformar comentários de clientes em insights claros, organizados e úteis para tomada de decisão.

---

# 🧠 Contexto

O prompt foi desenvolvido para análise de feedbacks relacionados a:

- Aplicativo bancário
- Pix
- Cartão de crédito
- Atendimento via chat

Os resultados da análise podem apoiar equipes de experiência do cliente na melhoria dos canais digitais e na redução de problemas recorrentes.

---

# 📂 Dados Utilizados

Os feedbacks analisados podem conter:

- Data do comentário
- Canal de atendimento
- Texto do feedback
- Produto citado
- Nota de satisfação (1 a 5)

---

# ⚙️ O que a IA Deve Fazer

A IA deve:

- Classificar feedbacks por tema, sentimento, urgência e produto citado
- Identificar padrões e problemas recorrentes
- Destacar elogios e oportunidades de melhoria
- Utilizar evidências presentes nos comentários
- Sugerir ações práticas para os times responsáveis

---

# 📄 Formato Esperado da Resposta

O prompt orienta a IA a gerar:

## ✅ Resumo Executivo
Resumo curto com os principais insights encontrados.

## ✅ Tabela de Análise
Tabela contendo:

- Tema
- Sentimento
- Evidência
- Ação sugerida

## ✅ Lista de Prioridades
Lista com as 3 prioridades mais importantes para melhoria da experiência do cliente.

---

# ⚠️ Restrições Definidas

O prompt estabelece os seguintes cuidados:

- Utilizar apenas os dados fornecidos
- Não inventar números ou conclusões
- Não expor dados pessoais ou sensíveis
- Informar limitações quando necessário
- Utilizar linguagem simples e objetiva
